# branch
```shell
git branch

git branch -v

# 删除本地分支
git branch -d branchName
git branch -D branchName

# 删除远程分支
git push origin --delete originBranchName

```

```shell
git remote -v

git remote add czp git@github.com:czp-first/ToBeBetter.git
```
